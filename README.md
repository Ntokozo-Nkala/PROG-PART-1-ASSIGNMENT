# PROG-PART-1-ASSIGNMENT
THIS PROJECT REGISTERES A USER ACCOUNT THEN LOGS THE USER INTO THE REGISTERED ACCOUNT
THE PROGRAM THEREAFTER PROMTS THE USER FOR THE FOLLOWING:
- FIRST NAME
- LAST NAME
- USERNAME
- PASSWORD
- CELL PHONE NUMBER
  THE USER INPUTS WILL THEN NEED TO HAVE TO PASS THE CONDITIONS INSIDE THE METHODS
  IF ALL ARE PASSED THEN THE PROGRAM SUCCESSFULLY REGISTERES THE USER
  IF ONE OF MORE OF THE CONDITIONS ARE NOT MET, THE PROGRAM FAILS AND OUTPUTS AN APPROPRIATE ERROR MESSAGE.
  ONCE THE USER HAS BEEN SUCCESSFULLY REGISTERED, A SUCCESS MESSAGE IS THEN DISPLAYED
  THEN THE USER WILL BE ABLE TO LOGIN INTO THEIR ACCOUNT, IN THE FOLLOWING INFORMATION:
  - USERNAME
  - PASSWORD
    THERES A METHOD THAT VERIFIES THOSE ENTERED INPUTS. THE USERNAME AND PASSWORD MUST MATCH THE ONES ENTERED WHEN REGISTERING
   BUT IF THE USERNAME AND PASSWORD DO NOT MATCH THE ONES ENTERED WHEN REGISTERING, THE LOGIN WILL AUTOMATICALLY FAIL.
APPON A SUCCESSFUL LOGIN, THE PROGRAM WIILL THEREAFTER RETURN THE LOGIN STATUS AND A WELCOME MESSAGE WILL BE DISPLAYED WITH THE USERS FIRST NAME AND LAST NAME.
IF IT FAILS THE USER WILL THEN BE PROMPED TO TRY TO LOGIN AGAIN.
